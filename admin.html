Bağlan GITHUB TOKEN

Token yalnızca bu tarayıcıda saklanır, hiçbir sunucuya gönderilmez — panel doğrudan GitHub ile konuşur.

Token nasıl alınır?

github.com/settings/personal-access-tokens/new adresinden "Fine-grained token" oluştur: Only select repositories → Ciftci-Workshop → Repository permissions → Contents: Read and write. Oluşan token'ı buraya yapıştır.
